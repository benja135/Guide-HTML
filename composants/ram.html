La mémoire vive (RAM)

Comme pour tous les autres composants d'un ordinateur, il existe différents types de RAM.

Les différents types RAM

Chaque type de ram convient à un type de processeur.

DDR4

La DDR4 est le type de RAM le plus récent. En terme de capacité elle est équivalente à la DDR3, cependant la fréquence minimale de la DDR4 correspond à la plus elevé de la DDR3. Elle consomme également moins d'énergie. Elle est utilisée pour les Intel Core Socket 1151 et 2011-3.

DDR3

La DDR3 est encore le type de ram le plus utilisé aujourd'hui. Elle est utilisée pour:

  • les Intel Core Socket 1150, 1151, 1155, 1156, 1366.
  • les AMD Phenom II et Athlon II sur la carte mère AM3 et sur les sockets FM1 et FM2

    • DDR2

    Aujourd'hui ce type de RAM n'est plus vendu neuf car trop ancien. Cependant il est possible d'en trouver d'occasion.

    La vitesse

    Chaque catégorie de RAM est suivi de plusieurs évolutions impactant sur les performances. Ces évolutions se font principalement sur la fréquence avec laquelle la RAM est capable de fonctionner. Ainsi la mémoire DDR3 1800 (1800Hz) est plus récente et plus rapide que la DDR3 1333 (1333Hz).

    La latence

    La latence correspond au temps de réaction entre la réception de l'information et le traitement de celle-ci. Plus la fréquence est élevée, plus la latence augmente. Ainsi, plus le temps de latence est petit, plus la ram sera rapide.
