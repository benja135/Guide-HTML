Lexique

Vous avez déjà parcouru le site, mais vous ne comprenez pas un seul mot de ce qui est écrit ? Le lexique est fait pour vous ! Vous trouverez ci-dessous la liste des termes employées dans ce guide, ou communément dans le jargon informatique.

  • Alimentation : L'alimentation s'occupe de donner la puissance de votre ordinateur. La puissance est donnée en Watt (W).
  • BIOS : Le BIOS est le logiciel le plus simple installée sur la Carte Mère. Il permet de naviguer dans les options prédéfinies de la configuration de la Carte Mère.
  • Bit : Les informations dans votre ordinateur sont codées en bit. Un bit a deux valeurs possibles : 0 ou 1. On parle souvent de langage binaire.
  • Boitier : Le boitier contient l'ensemble des composants de l'ordinateur.
  • Bus : Les bus transportent les données entre les composants de l'ordinateur.
  • Carte Graphique (GPU) : La carte graphique est le composant permettant de faire tous les calculs d'affichage.
  • Carte Mère (MB) : La carte mère est le composant acceuillant tous ls autres composants. C'est la base où va se connecter le processeur, carte graphique, etc...
  • Clavier : Le clavier est ce qui vous permet de saisir des informations sur votre pc. Les claviers avaient autrefois des prises US2, mais la plupart des claviers se branchent maintenant en USB.
  • Chipset :
  • Disque Dur : Le disque dur est ce qui contient les informations de votre ordinateur, c'est la mémoire. Il existe des disques dur HDD et SDD.
  • DVI : Le DVI est un port de branchement audio-visuel.
  • HDMI : Le HDMI est un port de branchement audio-visuel de haute qualité, apportant une image pouvant aller à 1080p.
  • Hertz (Hz) : C'est l'unité de mesure de la fréquence. On en parle notamment de Hz pour le CPU, exprimé en GHz. Un GHz = 1 000 000 000Hz
  • Octet (o) : Un octet correspond à 8 bits.
  • Operating System (OS) : Interface graphique permettant d'intéragir avec l'ordinateur.
  • PCI : Port permettant d'installer la GPU, différentes cartes, etc...
